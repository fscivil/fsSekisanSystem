# fsSekisanSystem
Excel/VBAで作成した積算支援ツール fs積算System の公開リポジトリ

＃fs積算System（Ver11.3）

　Excel/VBA を用いて作成した、土木工事向けの積算支援ツールです。  
　数量計算書の作成、内訳書の作成、予算書の作成、単価管理など、  
　積算業務に必要な作業を効率化するためのフリーソフトです。

## ■ 主な機能
 　　・数量計算書の作成（自動計算・自動整形）
   　・資材・外注の見積依頼可能（送信設定必要）
　　 ・予算書の作成
 　　・代価表・単価表データを代価項目実行に登録し、他積算へ再利用
 　　・工種別の集計機能
     ・内訳書の作成（Excel 形式で出力）
 　　・受注後には出来高調書・原価分析書作成し進捗管理可能
     ・令和8年7月より適用される 「材料費・労務費・法定福利費・建退共・安全衛生経費」 の計上方式に対応
 　　・VBA による自動化処理（高速・安定動作）

## ■ 動作環境
 　　・Windows 10 / 11  
　　 ・Microsoft Excel 2016 以降（VBA が使用可能な環境）

## ■ インストール方法
　　 1. 本リポジトリの **Releases** または **ZIP ファイル** をダウンロード  
　　 2. ZIP を任意のフォルダに展開  （C:¥fs積算）
　　 3. Excel マクロを有効にして起動  
　　 4. `fs積算System.xlsm` を開くと利用できます
